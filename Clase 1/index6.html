Curso de desarrollo Frontend

Bienvenido al curso de desarrollo Frontend. En este curso, vas a aprender a crear la interfaz visible y la experiencia de usuario (UI/UX) de sitios web y aplicaciones. Los temas clave incluyen el uso de HTML para la estructura, CSS para el diseño y la adaptabilidad visual (responsividad), y JavaScript para la interactividad. También se cubren frameworks y bibliotecas populares como React, que facilitan la creación de interfaces de usuario dinámicas y eficientes. Al finalizar el curso, los estudiantes estarán equipados con las habilidades necesarias para diseñar y desarrollar sitios web modernos y atractivos.

  1. Introducción a HTML
  2. Introducción a CSS
  3. Introducción a Sass
  4. Introducción a JavaScript
  5. Introducción a React
